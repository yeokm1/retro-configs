Xircom CardBus Ethernet PC Card
Specifications

The Xircom CardBus Ethernet PC Card provides connection to 10Mbps or 100Mbps Ethernet networks for access to network utilities, the Internet, E-mail and other network resources.

Specifications:

Cable and Connector:
100Base-TX unshielded twisted pair for use on category 5 (data grade) cabling; 10Base-T unshielded twisted pair (UTP)
Connector Type: RJ-45
Operating Distance: 328 ft. (100m)
Certification: FCC Part 15, Class B, CE Mark, Class B
Ethernet Standards: IEEE 802.3 for 10Mbps, 802.3u for 100Mbps
Non-volatile RAM: LAN adapter ID and serial number
LEDs:
Transmission activity (yellow)
Link (green) or power (red)
Operating System Compatibility:
Supports Windows 95 (OSR2) or later, Windows Me (Millennium Edition), Windows 98, Windows NT 4.0 and Windows 2000, using Xircom drivers supplied
Network Operating System Compatibility: Supports all popular network operating systems including Novell NetWare, Microsoft Windows 95/98, Windows Me, Windows NT, Windows 2000, Windows 3.x, Microsoft LAN Manager, Artisoft LANtastic, Banyan VINES, DEC Pathworks and IBM OS/2
