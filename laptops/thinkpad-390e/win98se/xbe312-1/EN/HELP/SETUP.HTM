Xircom CardBus Ethernet PC Card
Installation

Note: The technical information in this online documentation is valid for the following Xircom products: RealPort2 CardBus Ethernet 10/100, RealPort CardBus Ethernet 10/100, CardBus Ethernet II 10/100, and CardBus Ethernet 10/100.

This section provides the following information and instructions:
  • Installing Hardware
  • Installing Software

Installing Hardware

  • Plug the Xircom CardBus Ethernet PC Card into the PC Card slot on your computer, and connect cables, as shown in the printed Installation Guide.
  • Proceed to the software installation instructions for your operating system:

Installing Software

Driver installation procedures vary for each operating system. The following operating systems are supported:

  • Windows 2000
  • Windows Me (Millennium Edition)
  • Windows 98
  • Windows 95
  • Windows NT 4.0

Windows 2000

Note: Windows 2000 may automatically install its own built-in driver for your Xircom CardBus Ethernet PC Card. If an automatic installation occurs when you insert the Xircom PC Card, follow the onscreen instructions to complete the installation, then see Windows 2000 Troubleshooting for information on how to upgrade to the latest driver version, using the Xircom compact disc. If no automatic installation occurs, follow the New Hardware Wizard instructions that follow:
  1. When you insert the Xircom CardBus PC Card, the Found New Hardware message will appear, followed by the Found New Hardware Wizard dialog. Click Next.
  2. On the Install Hardware Device Drivers screen, your CardBus PC Card will be identified (for example, you might see "RealPort2 CardBus Ethernet 10/100" or "Xircom CardBus Ethernet 10/100"). Select "Search for a suitable driver for my device (recommended)" and click Next.
  3. On the Locate Driver Files screen, place a check in  the "Specify a location" check box. All other options should be cleared (unchecked). Click Next.
  4. In response to the message "Insert the manufacturer's installation disk...," insert the Xircom compact disc and type the path to the Xircom compact disc (for example, if your CD-ROM drive is E, type E:\), then click OK.
  5. On the Driver Files Search Results screen, click Next again to install the driver displayed. On the next screen, click Finish. If you receive a Digital Signature Not Found message, click Yes to continue the installation. If you click No the installation will be terminated.
    6.
  6. Click Start, and then Run. Type the path to the SETUP.EXE program on the Xircom compact disc (for example, if your CD-ROM drive is E, type E:\SETUP.EXE), then click OK. 
  7. On the menu that appears, click "Install Xircom Software and Tools" and follow the onscreen instructions. Xircom online documentation and utilities will be copied to your hard drive. To access Xircom utilities or view online documentation after running SETUP, click Start, Programs, Xircom Utilities. You must have Microsoft Internet Explorer or Netscape Navigator installed to view the online documentation.
  8. Click Yes if prompted to restart the computer.
  9. After Windows reloads, you can log on to the network.

Windows Me (Millennium Edition)

  1. When you insert your Xircom CardBus Ethernet PC Card, Windows Me (Millennium Edition) will automatically install a built-in Windows driver for the Xircom card. Follow the onscreen instructions displayed by Windows to complete the installation, then see Windows Millennium Troubleshooting for instructions on how to upgrade to the latest Xircom driver version shipped on the Xircom compact disc. Xircom recommends that you upgrade to the latest Xircom drivers to take advantage of the full functionality of your Xircom card.
    NOTE: To install Xircom utilities and online documentation onto your hard drive after an automatic installation, perform steps 2 and 3 below.
  2. Click Start, and then Run. Type the path to the SETUP.EXE program on the Xircom compact disc (for example, if your CD-ROM drive is E, type E:\SETUP.EXE), then click OK. 
  3. On the menu that appears, click "Install Software and Tools" and follow the onscreen instructions. Xircom online documentation and utilities will be copied to your hard drive and installed under Start, Programs, Xircom Utilities. You must have Microsoft Internet Explorer or Netscape Navigator installed to view the online documentation.

Windows 98

  1. When you insert the Xircom CardBus PC Card, the Add New Hardware Wizard appears.
  2. Insert the Xircom compact disc and click Next.
  3. Choose "Search for the best driver..." and click Next.
  4. Select "CD-ROM drive" and click Next.
  5. Click Next again to install the driver displayed. Click Finish. 
  6. Click Start, and then Run. Type the path to the SETUP.EXE program (for example, if your CD-ROM drive is E, type E:\SETUP.EXE), and then click OK. 
  7. From the menu that appears, select "Install Xircom Software and Tools" and follow the onscreen instructions. Xircom online documentation and utilities will be copied to your hard drive. To access Xircom utilities or view online documentation after running SETUP, click Start, Programs, Xircom Utilities. You must have Microsoft Internet Explorer or Netscape Navigator installed to view the online documentation.
  8. Click Yes if prompted to restart the computer.
  9. After Windows reloads, you can log on to the network.

Windows 95

  1. When you insert the Xircom CardBus PC Card, the Update Device Driver Wizard appears. 
  2. Insert the Xircom compact disc and click Next. 
  3. Accept the driver detected and click Finish. If this is your first network installation, you will be asked to type in a unique name for your computer and a network workgroup name which should be obtained from your network administrator. 
  4. If prompted for additional files, click OK, and then type the path to the Xircom compact disc (for example, if your CD-ROM is E, type E:\) and click OK. 
  5. Click Start, and then Run. Type the path to the SETUP.EXE program on the Xircom compact disc, then click OK. 
  6. From the menu that appears, select "Install Xircom Software and Tools" and follow the onscreen instructions. Xircom online documentation and utilities will be copied to your hard drive. To access Xircom utilities or view online documentation after running SETUP, click Start, Programs, Xircom Utilities. You must have Microsoft Internet Explorer or Netscape Navigator loaded to view the online documentation.
  7. Click Yes if prompted to restart the computer.
  8. After Windows reloads, you can log on to the network.

Windows NT 4.0

  1. Start Windows NT 4.0 and log on as Administrator.
  2. Connect cables and insert the Xircom CardBus Ethernet PC Card into the PC Card slot as shown in the printed Installation Guide.
  3. Insert the Xircom CardBus Ethernet PC Card compact disc into your CD-ROM drive.
  4. Select Network from the Control Panel.
  5. Select the Adapters tab and click Add.
  6. On the Select Network Adapters screen, click Have Disk. Insert the Xircom compact disc and type the path to the CD-ROM (for example, if your CD-ROM drive is E, type E:\). Click OK.
  7. Select "Xircom CardBus Ethernet 10/100," and click OK.
  8. At the Xircom Ethernet Adapter Setup screen, choose appropriate settings. Click OK.
  9. At the Network screen click Close.
  10. At the Network Settings Change screen, click Yes. The computer will shut down and restart.
  11. After Windows reloads, run SETUP from the Xircom compact disc. Select "Install Xircom Software and Tools" and follow the onscreen instructions. Xircom online documentation and utilities will be copied to your hard drive. To access Xircom utilities or view online documentation after running SETUP, click Start, Programs, Xircom Utilities. You must have Microsoft Internet Explorer or Netscape Navigator  installed to view the online documentation.
